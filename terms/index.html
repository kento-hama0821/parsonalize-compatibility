第1条 定義

1. 「ユーザー」
本サービスを利用するすべての方。
2. 「診断データ」
ユーザーが入力した回答および MBTI® 結果・相性結果などの分析データ。
3. 「プレミアムユーザー」
有料プランを購入し追加機能を利用するユーザー。
4. 「コンテンツ」
本サービス上で提供される文章・画像・解析アルゴリズムその他一切の情報。

第2条 規約の適用

2.1 ユーザーは本規約に同意のうえ本サービスを利用するものとします。

2.2 未成年のユーザーは法定代理人の同意を得たうえで利用してください。

2.3 当社が別途定めるガイドライン等は本規約の一部を構成します。

第3条 ユーザー登録

3.1 メールアドレスによる会員登録またはソーシャルログインが可能です。

3.2 登録情報に虚偽があった場合、当社は登録を拒否・抹消できます。

第4条 サービス内容と変更

4.1 本サービスは MBTI® をベースとした性格診断と、ユーザー同士または著名人との相性診断結果を提供します。

4.2 当社は機能改善・システム保守のため、事前通知なくサービス内容を変更・停止することがあります。

第5条 アカウント管理

5.1 パスワードはユーザー自身の責任で管理してください。

5.2 不正ログインのおそれを確認した場合は直ちに当社へご連絡ください。

第6条 禁止行為

  1. 法令・公序良俗に違反する行為
  2. 診断結果の改ざん・無断転載・販売
  3. 解析アルゴリズムの逆コンパイル・リバースエンジニアリング
  4. 当社サーバーに過度な負荷を与える行為
  5. その他当社が不適切と判断する行為

第7条 プレミアムプラン

7.1 プレミアムユーザーは「詳細相性レポート」「過去データ比較」「週次アドバイスメール」等の機能を利用できます。

7.2 料金はウェブサイトに表示する方法で決済します。既払金は原則返金いたしません。

第8条 知的財産権

8.1 本サービスに関する一切の知的財産権は当社または正当な権利者に帰属します。

第9条 免責事項

9.1 本サービスの診断・アドバイスは心理的・医学的助言を提供するものではなく、参考情報としてご利用ください。

9.2 当社はシステム障害・通信回線のトラブル等により生じた損害について責任を負いません。

第10条 利用停止・登録抹消

当社はユーザーが本規約に違反した場合、事前通知なくアカウントを停止・削除できます。

第11条 個人情報の取扱い

当社はプライバシーポリシーに従いユーザーの個人情報を適切に取り扱います。

第12条 規約の変更

当社は必要に応じて本規約を改定できます。重要な変更はウェブサイトまたはメールで告知します。

第13条 準拠法・管轄裁判所

本規約は日本法に準拠し、東京地方裁判所を第一審の専属的合意管轄裁判所とします。

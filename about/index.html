Parsonalize 相性診断

サービスについて

Parsonalize 相性診断 は、16 タイプ MBTI 分析をベースに
「私（タイプA）とあの人（タイプB）の相性は？」 を 5 秒で可視化する 無料ウェブアプリです。星評価・レーダーチャート・強み／注意点の 3 本柱で、 「どう向き合えば関係が深まるのか」 を直感的に理解できます。

恋人・友人・同僚・推し… 相手をもっと知りたいときにご活用ください。 結果はクリック一つで共有リンク化できるので、SNS やチャットで送り合うだけで盛り上がります！

サービスコンセプト

「違い」を恐れず、「違い」を楽しむ。

人間関係においてすれ違いはつきもの。しかし タイプの特徴 をあらかじめ知っていれば、 行動や考え方の背景をポジティブに解釈できます。Parsonalize は “知る → 受け入れる → 活かす” の 3 ステップで、 あなたのコミュニケーションをアップデートします。

秒速レポート

2 つのタイプを選ぶだけで、星評価・平均パラメータ・解説文を即生成。

レーダーチャート

論理性・感受性など 6 軸で両者のバランスをひと目で把握。

ワンクリック共有

X / LINE / URL コピーに対応。診断結果をそのままシェアして盛り上がろう！
